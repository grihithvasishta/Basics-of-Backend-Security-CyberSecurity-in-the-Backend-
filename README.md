# 🔐 CyberVault — Backend Security Mastery

> A complete, self-contained cybersecurity learning guide for backend systems. Beginner-friendly, zero dependencies, runs in any browser.

---

## 📖 What Is This?

CyberVault is a single-file interactive HTML book that teaches backend security from the ground up. No frameworks, no build tools, no installs — just open `index.html` in your browser and start reading.

It's written for developers who want to understand **why** security works the way it does, not just memorize rules. Every concept comes with real-world analogies, code comparisons, and interactive simulators you can play with directly in the page.

## 🚀 Getting Started

1. Click the green **Code** button at the top of this page
2. Select **Download ZIP**
3. Unzip the file
4. Open `index.html` in your browser

That's it. No installs, no terminal, no setup.

---

## ✨ Features

- **100% self-contained** — one HTML file, no external dependencies at runtime
- **Interactive simulators** — try SQL injection, password hashing, and rate limiting live in the browser
- **Sidebar navigation** — jump to any chapter instantly
- **Dark theme** — easy on the eyes for long reading sessions
- **Mobile & tablet friendly** — readable on any screen size
- **No tracking, no ads, no accounts** — just content

---

## 🎯 Who Is This For?

- Junior developers learning web security for the first time
- Backend engineers who want to sharpen their security instincts
- CS students preparing for security interviews or coursework
- Anyone curious about how attacks like SQL injection or CSRF actually work

Prior coding experience is helpful but not required. Every concept is explained from first principles.
